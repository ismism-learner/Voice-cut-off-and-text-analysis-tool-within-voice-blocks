## 方式2: 使用真实音频（进阶）

### 前置要求

1. **安装FFmpeg**
```bash
# Ubuntu/Debian
sudo apt-get install ffmpeg

# macOS
brew install ffmpeg

# Windows
# 从 https://ffmpeg.org/download.html 下载
```

2. **安装Python依赖**
```bash
pip install -r requirements.txt
```

3. **配置API密钥**（可选，不配置则使用Mock模式）
```bash
cp .env.example .env
# 编辑.env文件，填入API密钥
```

### 运行GUI应用

```bash
python main.py
```

界面操作：
1. 点击"选择视频/音频文件"
2. 选择你的视频或音频文件（支持MP4, MP3, WAV等）
3. 点击"开始处理"
4. 等待处理完成（查看实时进度）
5. 切换标签页查看结果
6. 点击"导出结果"保存

## 测试案例场景

### 场景1: 哲学讲座分析

**输入**: 1小时的哲学讨论视频

**处理流程**:
```
视频 → 提取音频 → 检测停顿(1.5秒) → 分割为~100段
→ 语音转文字 → 识别"但是"、"因此"等标记词
→ 二次分段 → 分析逻辑关系 → 提取核心观点
→ 生成结构化报告
```

**输出**:
- 按时间轴排列的段落列表
- 段落间的逻辑关系图谱
- 核心论点 vs 支撑论据
- 主题树状结构

### 场景2: 会议记录整理

**输入**: 30分钟的项目讨论录音

**处理**:
- 自动识别讨论要点
- 区分主要决议和讨论过程
- 识别转折点和关键结论

**输出**:
- 结构化会议纪要
- 决策点标注
- 待办事项提取

## 测试数据示例

### 模拟音频内容（test_mock.py中）

```
段落1 (0-8秒):
"我们首先要理解什么是现象学。现象学是研究意识的本质结构。"

段落2 (8-16秒):
"但是这个概念很抽象，需要从不同角度来理解。比如胡塞尔的观点。"
→ 检测到标记词: "但是"（转折）、"比如"（举例）

段落3 (16-24秒):
"胡塞尔认为意识总是关于某物的意识。这是意向性理论的核心。"

段落4 (24-32秒):
"回过头来讲，现象学的目标是回到事物本身。"
→ 检测到标记词: "回过头来讲"（回溯）

段落5 (32-40秒):
"因此，我们需要悬置自然态度。这叫做现象学还原。"
→ 检测到标记词: "因此"（因果）

段落6 (40-48秒):
"然而，这种方法论并非没有争议。海德格尔就提出了不同看法。"
→ 检测到标记词: "然而"（转折）

段落7 (48-55秒):
"总之，现象学为我们提供了理解意识的新方法。"
→ 检测到标记词: "总之"（总结）
```

                                                    ↑

                                                    │

**主题分类**:

## 常见问题

### Q1: 测试脚本运行失败？

**检查Python版本**:
```bash
python --version  # 需要3.8+
```

**如果缺少依赖**:
```bash
pip install -r requirements.txt
```

### Q2: 想使用自己的音频文件但没有API密钥？

可以在配置文件中使用Mock模式：
```yaml
# config/api_config.yaml
stt:
  provider: "mock"  # 使用模拟模式
llm:
  provider: "mock"  # 使用模拟模式
```

这样可以测试音频分段功能，但转录结果会是模拟文本。

### Q3: 如何获取API密钥？

**阿里云语音识别**:
1. 访问 https://www.aliyun.com/
2. 注册账号并登录
3. 开通"智能语音交互"服务
4. 在控制台获取API Key

**通义千问**:
1. 访问 https://dashscope.aliyun.com/
2. 注册并登录
3. 获取API Key

### Q4: 处理1小时视频需要多久？

**取决于模式**:
- 本地Mock模式: 几秒钟（仅分段）
- 真实API模式: 5-10分钟（含转录和分析）

### Q5: 成本如何？

1小时音频约 ¥3-6（包括STT + LLM分析）
